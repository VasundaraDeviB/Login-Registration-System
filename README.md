# Login & Registration System

## Description
A simple C++ console application that allows a user to register and log in using a username and password.

## Features
- User Registration
- User Login
- Password Verification
- Menu-driven Interface
- Input Validation
- Functions
- Switch-case Menu

## Technologies Used
- C++
- Visual Studio Code
- g++

## How to Run

Compile:

```bash
g++ Login_Registration.cpp -o login
```

Run:

```bash
.\login
```
