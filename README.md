# Login & Registration System

## Description
This is a C++ console application that allows users to register and log in using a username and password. The program stores user credentials in a text file, prevents duplicate usernames, and verifies user login using file handling.

## Features
- User Registration
- User Login
- File Handling using `fstream`
- Stores user credentials in `users.txt`
- Prevents duplicate usernames
- Verifies login credentials from the file
- Menu-driven interface
- Displays appropriate success and error messages

## Technologies Used
- C++
- Visual Studio Code
- MSYS2 (g++)
- File Handling (`fstream`)

## How to Run

Compile the program:

```bash
g++ Login_Registration.cpp -o login
```

Run the program:

```bash
.\login
```

> **Note:** If Windows blocks newly created executable names on your system, you can compile with another output name (for example `cgpa`) and run that executable instead.
